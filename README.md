# ScheduleAI

ScheduleAI is a scheduling platform concept that reduces repetitive schedule-building work by capturing preferences once and reusing them later.

This repository currently contains an **early website prototype** (static HTML + Tailwind) with:
- University selection portal
- Student preference chat flow (demo)
- Admin portal (demo login) + mock insights

## Tech (current)
- HTML / CSS / JavaScript
- Tailwind CSS (CDN)

## Run locally
### Option 1: Open directly
Double-click `index.html`.
