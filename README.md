# ScheduleAI

ScheduleAI is an AI-powered student scheduling assistant that helps learners create balanced course, work, and study plans with mood-aware support and chat-driven adjustments.

This repository has been updated to a modern Next.js prototype with:
- React + Next.js App Router
- TypeScript
- Tailwind CSS
- Starter landing page and scheduler concept UI

## Run locally
1. Install dependencies:
```bash
npm install
```
2. Start the development server:
```bash
npm run dev
```
3. Open `http://localhost:3000`

## Project structure
- `src/app/` — Next.js application pages and layout
- `src/components/` — reusable UI components
- `package.json` — scripts and dependencies
- `tailwind.config.ts` — Tailwind CSS configuration

## Notes
- `index.html` remains in the repo as the legacy prototype landing page.
- Use the Next.js app for the current starter experience.
