# ScheduleAI

ScheduleAI is an AI-powered student scheduling assistant that helps students create balanced schedules for classes, work, studying, and personal time. The goal of this project is to reduce student stress by giving users a smarter way to plan their day and adjust their schedule using AI support.

---

## Project Overview

Students often have to manage many responsibilities at the same time, including classes, assignments, work shifts, commuting, studying, and personal life. ScheduleAI is designed to help students organize these responsibilities in one place.

The current version is a modern Next.js prototype with a landing page and starter scheduler UI. Future versions can include AI-generated schedules, chat-based schedule changes, mood-aware planning, user accounts, and calendar integration.
